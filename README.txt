В этом калькуляторе релизовано почти все, что требовалось. Не добавлено только поддержка дробей.

Мануал:

Поддерживается унарный минус:
    Пример: -(1+1)  Результат: -2.0
            -(-2)   Результат: 2.0

Поддерживается произвольная вложенность скобок:
    Пример: (((54+43)/3.3)+28/7-(7*3))  Результат: 12.393939...

Поддерживается сравнивание двух выражений:
    Пример: (27/3)*2 < (14/7)*2   Результат: false
            (27/3)*2 < (14/7)*10  Результат: true

Поддерживается  тернарный оператор:
    Пример:  10 > 5 ? 1 : 2  Результат: 1.0
             5 < (7*0.5) ? 12*(0.7+1) : (((54+43)/3.3)+28/7-(7*3))  Результат: 12.393939...
             5 == 5 ? 22:1  Результат: 22.0
             5 != 5 ? 22:1  Результат: 1.0
